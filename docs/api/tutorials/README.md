## Prerequisites

### Rust
- Rust 1.70+ installed
- Basic Rust knowledge
- Cargo for dependency management

### C/C++
- C11 or C++17 compiler
- HEDL shared library (.so/.dll/.dylib)
- CMake or Make for build configuration

### JavaScript/TypeScript
- Node.js 16+ or modern browser
- npm or yarn
- Basic TypeScript knowledge (recommended)

### MCP Server
- Rust toolchain or pre-built binary
- Understanding of JSON-RPC
- AI/LLM system supporting MCP
